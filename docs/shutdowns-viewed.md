# PATCH /shutdowns/{id}/viewed

Отметка остановки линии как просмотренной диспетчером.

## Метод

`PATCH`

## URL

`/shutdowns/{id}/viewed`

## Параметры URL

| Параметр | Тип   | Описание      |
|----------|-------|---------------|
| id       | int64 | ID остановки линии |

## Доступ

Роли: `sc`, `rais`

## Тело запроса

Не требуется.

## Успешный ответ

**Код:** `200 OK`

**Тело:**

```json
{}
```

## Ошибки

| Код | Описание                  |
|-----|---------------------------|
| 400 | Неверный формат ID        |
| 404 | Остановка линии не найдена |
| 500 | Внутренняя ошибка сервера |

## Пример

### Запрос

```bash
curl -X PATCH \
  'https://api.example.com/shutdowns/123/viewed' \
  -H 'Authorization: Bearer <token>'
```

### Ответ

```json
{}
```
